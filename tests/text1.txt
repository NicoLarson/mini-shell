    Par défaut

        - GetCurrentDirectory pour connaître le répertoire courant,
        - SetCurrentDirectory pour changer le répertoire courant,
        - GetSystemDirectory pour connaître le répertoire système de Windows (C:\Windows\System),
        - GetWindowsDirectory pour connaître le répertoire d'installation de Windows (C:\Windows)

        Pour devenir bon en C : Cours et tutoriels langage C : http://c.developpez.com/cours/
